# Globos. Ukrainian educational web platform.

The project is under developing.


## Server Requirements:

- PHP >= 5.5.9
- OpenSSL PHP Extension
- PDO PHP Extension
- Mbstring PHP Extension
- Tokenizer PHP Extension

## How to use:

--- 

    composer install 
    npm install 
    gulp 
    php artisan migrate
    make settings in .env and config/database.php files

---
